hexo-theme-vivia

一款为 Hexo 开发的主题，基于 Landscape 修改。 (开发中)

预览

💻在线预览

安装

  1. 通过 npm 安装主题。

    npm install hexo-theme-vivia

  2. 将配置文件 node_modules/hexo-theme-vivia/example_config.vivia.yml 复制至 Hexo 根目录并重命名为 _config.vivia.yml。

  3. 编辑 Hexo 配置文件 _config.yml，设置 vivia 为主题。

    theme: vivia

  4. 创建 about 页面

    hexo new page about
